Nível 1

Por que Estrutura de Dados começa falando de C?

Antes de codar, você precisa entender por que a disciplina liga dados, memória, processador, computadores e microcontroladores.

Bloqueado

Explicação lúdica

Imagine dois mundos: um computador moderno é como uma cozinha industrial enorme, cheia de máquinas, espaço e recursos. Um microcontrolador é como uma cozinha minúscula de food truck: dá para fazer muita coisa, mas você precisa economizar espaço, energia e movimento.

Estrutura de Dados entra exatamente aí: em qualquer plataforma, os dados precisam ser recebidos, guardados, processados e devolvidos. A diferença é quanto recurso você tem para fazer isso.

Explicação técnica

Os PDFs começam comparando computadores ricos em recursos com dispositivos simples. Processadores CISC possuem conjunto de instruções mais complexo, são versáteis e programáveis, mas caros. Dispositivos simples, como uma calculadora de 8 bits, podem usar microcontroladores RISC, com instruções mais limitadas e menor custo. Em microcontroladores, recursos como memória e processamento são restritos, então a forma de manipular dados precisa ser mais controlada.

Tradução didático → técnico

Jeito didáticoJeito técnico de faculdade
“Computador tem muita folga.”Computadores modernos possuem alta disponibilidade de memória, armazenamento e processamento.
“Microcontrolador é mais limitado.”Microcontroladores possuem menos memória, menor poder de processamento e conjunto de instruções mais restrito.
“C é mais cru.”C permite manipulação mais explícita de variáveis, tipos, endereços e estruturas de dados, sem abstrações nativas como classes, objetos, listas, collections, eventos ou GUI.

Linguagem C: compilação, main, argc, argv e esqueleto mental

A base que você precisa saber antes de escrever qualquer exercício.

Bloqueado

História da linguagem C

C é uma linguagem compilada, imperativa e de propósito geral, padronizada pela ISO, aplicável tanto a CISC quanto a RISC. Foi criada por Dennis Ritchie, na AT&T Bell Labs, entre 1969 e 1973, a partir de uma linguagem anterior dele mesmo chamada "B". Ritchie também desenvolveu o UNIX junto com Ken Thompson.

Frase de prova: C é uma linguagem de baixo nível com traços de alto nível, aproximando-se mais da linguagem humana que a linguagem de máquina. A versão atual é a C23.

Do PDF para o código

Programas em C ficam em arquivos .c, tradicionalmente com arquivo principal chamado main.c. Por ser compilada, C precisa ser convertida para linguagem de máquina. O PDF cita o compilador original cc, o gcc como compilador popular e o MinGW como opção comum no Windows.

Variáveis

Para declarar uma variável em C, primeiro vem o tipo, depois o identificador.

int numero;
float valor;

int n1, n2, n3;

Variável é um espaço de memória nomeado. O tipo define como aquele espaço será interpretado.

Constantes com const

const float PI = 3.14159;
const float EULER = 0.57721;

Uma constante nomeia um valor usado várias vezes. Com const, há tipo associado e verificação pelo compilador.

Constantes com #define

#define PI 3.14159
#define EULER 0.57721

#define é uma diretiva de pré-processador: substitui o identificador em tempo de compilação. Não possui tipo próprio nem verificação de tipo como uma constante.

Atenção: o slide mostra #define PI = 3.14159;, mas em C usual o formato correto é sem = e sem ;: #define PI 3.14159.

printf: saída de dados

printf(string) imprime no terminal. A string pode conter especificadores de formato, substituídos pelos valores posteriores na mesma ordem.

printf("Ola mundo!");

scanf: entrada de dados

scanf(string, point*) lê dados do terminal e grava no endereço da variável. Por isso usamos & em variáveis comuns.

int numero;
printf("Digite um numero: ");
scanf("%d", &numero);

Tradução técnica: scanf precisa do endereço de memória do espaço que será preenchido.

Strings em C

C não implementa string nativa. Para texto, você declara array de caracteres com espaço suficiente.

#include <stdio.h>
int main(int argc, char const *argv[]) {

    printf("%s", palavra);
    return 0;
}

Para leitura de texto, o PDF reforça que se trabalha com array de caracteres. Para textos com espaço, veremos %[^\n] também na aula de structs.

Funções e prototipação

Se a função retorna algo, indique o tipo. Se não retorna, use void. Tradicionalmente, funções devem ser declaradas antes do main; se ficarem depois, faça o protótipo no cabeçalho.

int main(int argc, char const *argv[]) {

Bloqueado

Homogêneo = mesmo tipo

Estruturas homogêneas são alocações serializadas na memória. Elas guardam uma massa de dados do mesmo tipo e cada posição é indexada.

Frase de prova: Uma estrutura homogênea armazena múltiplos elementos do mesmo tipo, organizados em posições indexadas.

Dimensões

DimensãoNome conceitualExemplo
1 dimensãovetorint valores[10];
2 dimensõesmatrizchar jogoDaVelha[3][3];
N dimensõesmultidimensionaldouble cubo[5][5][5];

Para a linguagem, tudo se resume a arrays.

Arrays em C

int valores[10];
valores[0] = 10;
valores[2] = 11;
valores[9] = 1000;

printf("%d", valores[0]);

Ao declarar int valores[10], o computador aloca 10 posições na memória. Os índices começam em 0 e vão até quantidade total - 1, ou seja, 9.

Pegadinha: no exemplo do PDF, valores[1] é impresso sem ter sido atribuído. Isso pode revelar lixo de memória/valor indefinido. Isso é excelente para prova.

    printf("%d ", valores[i]);
}

A variável i controla a posição acessada.

Matrizes em C

char jogoDaVelha[3][3];
jogoDaVelha[0][0] = 'X';
jogoDaVelha[2][1] = 'O';

Para acessar matriz, você precisa informar exatamente os índices das dimensões. Uma matriz 3x3 tem 9 posições.

Laços aninhados em matriz

for (int i = 0; i < 3; i++) {

        char simbolo = (i+j) % 2 == 0 ? 'X' : 'O';
        jogoDaVelha[i][j] = simbolo;
    }
}

Uma variável de controle para cada dimensão: i para linha, j para coluna.

Dados heterogêneos: structs, typedef, strings, aninhamento e array de structs

Bloqueado

Heterogêneo = campos diferentes

Estruturas heterogêneas armazenam múltiplos valores diferentes em um único arranjo. Elas não são uma simples sequência serializada como um array; funcionam como um nó de propriedades.

Analogia do PDF: um prontuário médico com número, nome, data, telefone, médico, limitações e laudo.

Struct como molde

struct ProntuarioMedico {
    int numero;
    char nomeCompleto[255];
    char dataAbertura[10];
    char telefone[16];
    char laudoClinico[1024];
};

struct ProntuarioMedico prontuario;

A struct define o molde; a variável cria uma instância desse molde.

typedef struct

typedef struct {
    int numero;
    char nomeCompleto[255];
    char telefone[16];
} ProntuarioMedico;

ProntuarioMedico prontuario;

Com typedef, você transforma a struct em um tipo e não precisa escrever struct ProntuarioMedico toda vez.

Operador ponto

prontuario.numero = 101203;
printf("Prontuario Nº: %d", prontuario.numero);

O ponto acessa um atributo da variável struct.

Strings dentro de struct

Para preencher arrays de char dentro de structs, use strcpy da biblioteca string.h. Atribuição direta em arrays de caracteres não é permitida.

#include <string.h>
strcpy(prontuario.dataAbertura, "01/01/2023");
printf("%s", prontuario.dataAbertura);

scanf em campos numéricos

scanf("%d", &prontuario.numero);

Usa & porque é necessário passar o endereço do campo numérico.

scanf em arrays de char

scanf("%s", prontuario.dataAbertura);
scanf("%[^\n]", prontuario.nomeCompleto);

Não usa & porque o array já se comporta como ponteiro para o primeiro caractere. %[^\n] lê texto com espaços.

typedef struct {
    int DDD;
    char numero[10];
} Telefone;

typedef struct {

    char nome[255];

} Aluno;

aluno.telefone.DDD = 44;
strcpy(aluno.telefone.numero, "3268-5940");

Array de structs

Aluno alunos[15];
for (int i = 0; i < 15; i++) {
    printf("Nome: ");
    scanf("%[^\n]", alunos[i].nome);
    printf("RA: ");
    scanf("%d", &alunos[i].ra);

Memória, endereçamento, scanf e strings por referência

A base da Aula 3 antes de ponteiros.

Bloqueado

RAM e Von Neumann

A memória volátil/RAM armazena informações temporárias do sistema operacional e aplicativos. Ela é intermediária entre disco e processador porque é mais rápida que o armazenamento de massa.

A Arquitetura de Von Neumann permite armazenar programas no mesmo espaço de memória que os dados.

Unidades da memória

UnidadeDefinição no PDF
bit0 ou 1
byte8 bits
página4096 bytes
segmento256 páginas

Word, Byte e Nibble

Uma word (palavra de memória) se divide em 2 bytes, e cada byte se divide em 2 nibbles (4 bits cada). O operador & retorna o endereço do byte inicial da variável — mas o endereçamento sozinho não sabe quantos bytes o espaçamento ocupa: é responsabilidade do programador informar o tipo/formato para o computador saber quantos bytes ler.

Endereço absoluto, real e virtual

TipoIdeia
AbsolutoAssociado à localização física na memória central; usado pelo processador para acessar blocos da memória física.
RealConvertido para endereço absoluto por prefixação pelo hardware; há um endereço real para cada byte incorporado.
VirtualDisponível para cada aplicação; convertido para real e depois absoluto. Isola a memória entre programas.

Vendo endereço com &

int valor;
printf("Endereco em Hex: %p\n", &valor);
printf("Endereco em Dec: %d\n", &valor);

O operador & retorna o endereço do byte inicial da variável. %p é o formatador de ponteiro/endereço.

Tipo e tamanho importam

O PDF destaca que o endereço aponta para o byte inicial, mas o programa precisa saber quantos bytes considerar. Por isso o tipo e o formatador correto são responsabilidade do programador.

Exemplo: se você usa %c, o scanf entende caractere; se usa %d, entende inteiro.

scanf trabalha com endereçamento

char letra;
printf("Informe uma letra: ");
scanf("%c", &letra);
printf("A letra informada e: %c", letra);

O valor digitado é inserido diretamente no endereço de referência do espaço de memória. O preenchimento correto depende do formatador.

Strings, byte inicial e caractere nulo

Strings são arrays de caracteres. Muitas funções trabalham com o endereço do byte inicial e percorrem a sequência até encontrar \0, o caractere nulo.

char texto[20] = "Algoritmos e Logica";
printf("Texto: %s", texto);
//Por referência do byte inicial (mesmo resultado):
printf("Texto: %s", &texto);

O programa não imprime necessariamente todas as 20 posições; ele percorre até o limite lógico do texto, marcado por \0. Muitas funções preferem receber o endereço do byte inicial (referência) a receber a variável diretamente — o resultado é idêntico, pois o computador busca todos os bytes da sequência do mesmo jeito.

A parte pesada da Aula 3, com exemplos e pegadinhas.

Bloqueado

Definição

Ponteiros são espaços de memória responsáveis por armazenar o endereço de outros espaços. Podem ser variáveis, constantes ou parâmetros de funções.

Embora endereço possa ser entendido como número, usar ponteiro tem vantagens: indica o tipo do espaço apontado e permite operadores específicos.

Declaração

int* ponteiroA;
char* ponteiroB;
float* ponteiroC;

O tipo do ponteiro indica o tipo do espaço de memória referenciado.

Atribuição de ponteiros

int variavelA;
char variavelB;
float variavelC;

int* ponteiroA = &variavelA;
char* ponteiroB = &variavelB;
float* ponteiroC = &variavelC;

Pegadinha: atribua o endereço de acordo com o mesmo tipo. Se o ponteiro for incompatível, o computador calcula errado a quantidade de bytes referente ao espaço.

scanf com ponteiro

int valor;
int* p_valor = &valor;
printf("Informe um numero: ");
scanf("%d", p_valor);
printf("Endereco: %p | Valor: %d", p_valor, *p_valor);

Não usa & no scanf porque p_valor já guarda o endereço válido.

Recuperação com *

int variavelA = 10;
int* ponteiroA = &variavelA;
printf("%p = %d", ponteiroA, *ponteiroA);

*ponteiroA acessa o valor armazenado no endereço apontado.

Atribuição através do ponteiro

int variavelA = 10;
int* ponteiroA = &variavelA;

*ponteiroA = 25;
printf("%d", variavelA); // 25

Você altera o conteúdo do espaço apontado, sem usar diretamente o identificador original.

Passagem por referência

#include <stdio.h>
void funcao(char*);

int main(int argc, char const *argv[]) {
    char letra;
    funcao(&letra);
    printf("%c", letra);
    return 0;
}

void funcao(char* p) {
    *p = 'X';
}

A função recebe o endereço da variável e altera o valor original.

Prática de código

Crie uma função trocar que receba dois ponteiros de inteiro e troque os valores entre as variáveis.

void trocar(int* a, int* b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

int main() {
    int x = 10, y = 20;
    trocar(&x, &y);
    return 0;
}

Bloqueado

Ponteiros trabalhando com vetores

Ponteiros podem percorrer estruturas serializadas apenas incrementando ou decrementando seu valor. O computador calcula os bytes necessários conforme o tipo apontado.

char* retornarArray() {
    static char array[8] = {'Z', 'X', 'Q', 'A', '3', '6', '8', 'T'};
    return array;
}

int main() {
    char* p_array = retornarArray();
    for (int i = 0; i < 8; i++) {
        printf("%p: %c\n", p_array, *p_array);
        p_array++;
    }
    return 0;
}

p_array++ avança para o próximo caractere porque o ponteiro é char*.

Ponteiro de struct

typedef struct {
    int RA;
    char nome[255];
} Aluno;

Com variável normal, use ponto: aluno.RA. Com ponteiro para struct, use seta: p_aluno->RA.

Ponteiro de ponteiro

int valor = 3;
int* p_valor = &valor;
int** pp_valor = &p_valor;

printf("Ponteiro: %p\n", *pp_valor);
printf("Valor: %d", **pp_valor);

int** armazena o endereço de um ponteiro. *pp_valor acessa o ponteiro original; **pp_valor acessa o valor final.
